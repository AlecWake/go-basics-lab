# Go Basics Lab

This repository contains small Go programs used to explore and understand
core language behaviors before building larger backend systems.

The goal of this lab is to build intuition for how Go handles memory,
data structures, testing, and performance.

Each folder will demonstrate a specific concept.

Planned experiments include:

- Slice growth behavior (len vs cap)
- Map behavior and common pitfalls
- Pointer vs value receivers
- Table-driven tests
- Profiling and performance debugging with pprof
